Parachute video analysis¶

  1. 動画情報確認
    • フレーム数確認
    • フレーム幅,高確認
  2. 動画上のフレーム当たり秒数確認
    • 動画のフレーム率を確認し,フレーム当たり秒数を取り出す
  3. 動画上のピクセル当たりの実長確認
    • 動画の一フレームを取り出し,フレームあたら実長を取り出す
  4. 落下時間確認
    • 落下開始時刻(フレーム)確認
    • パラシュート展開時刻(フレーム)確認
    • 着地時刻(フレーム)確認
  5. 落下軌跡確認
    • 動画上に点を取り,軌跡を求める
  6. 時間-距離グラフ
    • xピクセル-yピクセル座標系
    • 時間[frame]-ピクセル座標系
    • 時間[s]-ピクセル座標系
    • 時間[s]-メートル座標系
  7. 時間-速度グラフ
    • 時間-速度グラフ
    • 終端速度
      • 十分減速した時点から地面到達までの平均速度
  8. 抗力係数を求める
    • F = ma
      • ma = mg -D
      • ma = mg - 0.5ρ(v^2)ACd

import cv2

def pause(i,min,max):
    if min<=i<=max:
        if i == min:
            print(f'{i} frame')
        if cv2.waitKey() == ord('p'):
            pass

cap = cv2.VideoCapture('1st front.mp4')
i = 1

std_coor = [(820,146),(820,890),(923,146)]

coral_pink = (128,128,240)

while cap.isOpened():
    ret, frame = cap.read()

    if not ret:
        break
    cv2.line(frame,std_coor[0],std_coor[2],coral_pink,1,cv2.LINE_AA)
    cv2.line(frame,std_coor[0],std_coor[1],coral_pink,1,cv2.LINE_AA)

    cv2.imshow('video',frame)

    pause(i,409,409) # 落下開始
    pause(i,434,434) # パラシュート展開
